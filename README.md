<h1 align="center">👨‍⚕️ Doctors-App</h1>

<p align="center">
  A modern platform for doctor appointments, scheduling, video consultations, and more.
</p>

---

## 🚀 Features

- 🔐 Authentication via Clerk  
- 👨‍⚕️ Patient & Doctor Dashboards  
- 📹 Video Calls with Vonage  
- 📅 Appointment Booking & Management  
- 🧑‍💼 Admin Panel for Clinic Management  
- ⚡ Built with modern tools (Next.js 15, Prisma, Tailwind, Neon)

## 🛠️ Getting Started

### 1️⃣ Clone the Repository

git clone https://github.com/AryanJiyani007/Doctors-App.git
cd Doctors-App

### 2️⃣ Install Dependencies

npm install

### 3️⃣ Configure Environment Variables

.env

DATABASE_URL=

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=

CLERK_SECRET_KEY=

VONAGE_API_KEY=

VONAGE_API_SECRET=

### 4️⃣ Push Prisma Schema to Database

npx prisma db push

### 5️⃣ Start the Development Server

npm run dev
